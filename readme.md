## 工作中封装的基于Vue开发的公用组件，单独建库
## 使用说明

``` bash
#克隆项目
git clone https://github.com/kannieXu/kannieUI.git

# 安装依赖
npm install

# 本地开发环境 访问http://localhost:8888
npm run dev
# 打包库
npm run npm
# 打包文档
npm run build
# 组件的UI测试
npm run test
```
